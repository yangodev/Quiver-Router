Your Quiver Router seems to work properly

Also try following routes:
  1. /api/table : shows your entire route table.
  2. /api/multistack: shows you the work of middleware.
  3. /api/param/simple/@value/: here a parameter is parsed.
  4. /api/param/multi/@valueN/@valueN+1/: here four parameters are parsed.
  5. /api/inherits/url and /api/inherits/id here middleware is inherited by url / id.
  6. /api/reverse/@id: this shows how to route reversing works.
  7. /api/cookie/single/@name/@value/: Cookie api included!
  8. /api/cookie/multi/…names/$/…values$/@maxAge: Comfortable API indeed!
  9. /static/path/to/file: static file serving works too.
  10. Templating support is also included. See the readme for more info.
